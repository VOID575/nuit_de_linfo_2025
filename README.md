# Nuit de l'info 2025 - Groupe GrumpyKangaroo
Le front-end est dans le dossier `frontend/`.

## Lien vers notre site
https://void575.github.io/nuit_de_linfo_2025/

Si vous voulez exécuter le projet `frontend/`, exécutez les commandes suivantes :
 - `npm install` ← Installation des dépendances NPM
 - `npm run dev` ← Démarrage du serveur en localhost avec Vite (port précisé dans le terminal)

# Où trouver nos défis ?
- Défi national : le site en lui même
- Défi Retour vers le passé : la douleur du web des années 2000 :  section NIRD (la route ne semble pas fonctionner)
- Défi ft_rube_goldberg : footer de la page principal
- Défi Hidden Snake : sur la page principale : pour lancer le snake executer la combinaison de touche suivante : Haut Haut Bas Bas Gauche Droite Gauche Droite A B
- Défi En trois dimension : snake en 3d et niveau 4 de la page
- Défi On veut du gros pixel : niveau 1
